PHP и JavaScript

Языки програмирования PHP и JavaScript JavaScript позволяют сделать сайт динамичным, то есть реагирующим на действия пользователя. Например, можно красивую выпадающую менюшку или слайдер.

Виды скриптов

Для этого пишуться скрипты(англ. script -"сценарий")- программы, позволяющие реагировать на действие пользователя, Скрипты бывают двух видов:

  • те,которые выполняются на сервере, а результат их выполнения приходит в браузер к пользователю уже в готовом виде. Это скрипты, написанные на языке PHP .На нем пишуться СМС-ки- системы управления контентом.
  • те, которые виполняються прямо в браузере пользователя. Это скрипты, написанные на языке JavaScript . Они чаще всего используються для, того чтобы сделать страницу более удобной и красивой.
